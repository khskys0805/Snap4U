# Snap4U

> 언제 어디서든 당신을 위해 스냅을 기록하세요! 사진 촬영 후 원하는 프레임을 선택하면 나만의 특별한 스냅을 남길 수 있습니다 :)

<br/>

## 기능 소개

### 📸 사진찍기

-   원하는 컷을 선택한 후 10장을 찍습니다.
-   찍은 사진 중 4장을 선택하여 원하는 프레임 색상을 고르면 이미지가 저장됩니다.

### 🖼 갤러리

-   촬영한 사진들을 갤러리에서 조회할 수 있습니다.
-   다운로드 버튼을 누르면 해당 이미지가 저장됩니다.
